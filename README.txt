Инструкция по запуску:

1.Перейти на страницу репозитория https://github.com/closeyoureyess/petRegisterOfEquipment
2.Кликнуть на зеленую кнопку Code
3.Выбрать пункт выгрузки зип-архива
4.Как только архи скачается, открыть его через винрар
5.Распаковать в любую удобную папку
6.Открыть IDEA
7.В верхней панели программы,выбрать File
8.Из выпадающего списка кликнуть на Open
9.Выбрать распакованную папку и дождаться, пока проект развернется
10.Как только все зависимости будут подгружены, перейти в main -> java -> PetRegisterOfEquipment.class
11.Запустить программу

Для просмотра документации, можно перейти на страницу http://localhost:8080/swagger-ui/index.html#/
